# ghcli v1.0.4

This release updates the public positioning for `ghcli` and the bundled
OpenClaw skill.

## Highlights

- Uses the human-facing name `Google Health (Fitbit) Local Archive`.
- Describes `ghcli` as a local-first Google Health (Fitbit) CLI and archive.
- Clarifies that `ghcli` uses the Google Health API for the same underlying
  health history.
- Describes the skill as reading Google Health API data for Fitbit/Google
  health accounts.
- Keeps the skill metadata requirement for the `ghcli` binary and updates the
  Go install stanza to `github.com/fdsouvenir/ghcli@v1.0.4`.

No sync, schema, OAuth, or query behavior changed in this release.
